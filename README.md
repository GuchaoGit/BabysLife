# BabysLife
关注儿童成长过程。
1、查看儿童身高体重比重表。  
2、记录基本信息。  
3、添加成长记录。  
4、实现数据备份与恢复。
5、生日提醒
6、BMI计算
7、添加身高、体重变化曲线
